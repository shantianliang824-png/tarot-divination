# 塔罗牌占卜

一个纯前端、离线可用的塔罗牌占卜 Web 应用。

## 功能

- 完整 78 张塔罗牌（22 张大阿尔卡纳 + 56 张小阿尔卡纳）
- 5 种占卜牌阵：单张、三张、凯尔特十字、爱情、事业
- 正位/逆位随机判定
- 卡牌翻转动画
- 自动解读分析
- 占卜历史记录（本地存储）
- 离线可用

## 使用方法

直接用浏览器打开 index.html 即可使用，无需服务器。

## 牌阵说明

单张占卜(1张) - 每日指引、快速解答
三张牌阵(3张) - 过去-现在-未来分析
凯尔特十字(10张) - 全面深度分析
爱情牌阵(5张) - 感情关系分析
事业牌阵(5张) - 职业发展分析

## 技术栈

原生 HTML + CSS + JavaScript
Web Storage API（本地数据存储）
无第三方依赖
