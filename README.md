# Tarot Divination · 塔罗占卜

3D 旋转牌圈选牌 · 纯前端离线可用

## 在线体验

https://shantianliang824-png.github.io/tarot-divination

## 功能

- 3D 牌圈交互：24 张牌排成圆环旋转，用户凭直觉选牌
- 78 张完整塔罗牌（22 大阿尔卡纳 + 56 小阿尔卡纳）
- 5 种牌阵：单张 / 三张 / 凯尔特十字 / 爱情 / 事业
- 正位 / 逆位随机判定
- 3D 翻牌动画 + 智能解读
- 占卜历史记录（浏览器本地存储）
- 完整离线可用

## 使用

在线访问或下载后双击 index.html
